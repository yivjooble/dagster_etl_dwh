#### Credentials format

Slack alert chat credentials
credentials/chat_credentials.json
```
{
  "chat_url": ""
}
```

DWH credentials
credentials/dwh_cred.json
```
{
  "user": "",
  "password": "",
  "host": "",
  "port": ,
  "database": ""
}
```

Pentaho credentials
credentials/pentaho_cred.json
```
{
  "user": "",
  "password": ""
}
```
