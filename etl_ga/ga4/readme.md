# GA4 ETL

## General info:
This project is to create ETL process for new GA4 (using apiv1 beta)

## Setup:
1. Create virtual env by:
    - `python -m venv venv`
2. Install needed requirements:
    - `pip install -r requirements.txt`
3. Activate virtual environment:
    - `venv\Script\activate`

## Usage
1. Run file:
    - `python main.py`
